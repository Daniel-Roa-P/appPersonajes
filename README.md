# App personajes

En el actual repositorio se encuentra una aplicación de Android en la cual se despliega un menú de personajes con la posibilidad de añadirle un arma, escudo y/o montura, los cuales se imprimirán en una imagen al presionar el botón construir.

En la lógica del proyecto hace uso de los patrones Abstract factory el cual proporciona productos con las imágenes en cuestión y el patrón Builder el cual construye una nueva imagen con los productos anteriores.

Para más información de la implementación de los patrones puede indagar en mi repositorio llamado Catalogo3.
